# CodeAlpha Data Science Internship

## Task 2:  Unemployment Analysis with Python

## Project Overview

This project was completed as part of the **CodeAlpha Data Science Internship**.

The objective of this project is to analyze unemployment trends using Python, explore the impact of the COVID-19 pandemic on unemployment rates, identify regional and seasonal patterns, and generate insights that can support economic and social policy decisions.

---

## Objectives

- Analyze unemployment rate data.
- Perform data cleaning and preprocessing.
- Explore unemployment trends using visualizations.
- Investigate the impact of COVID-19 on unemployment rates.
- Identify regional and seasonal unemployment patterns.
- Generate meaningful insights from the data.

---

## Dataset Information

The dataset contains unemployment statistics across different regions of India.

### Features

- **Region** – State/Region name
- **Date** – Observation date
- **Frequency** – Monthly frequency
- **Estimated Unemployment Rate (%)** – Percentage of unemployed individuals
- **Estimated Employed** – Number of employed individuals
- **Estimated Labour Participation Rate (%)** – Labour force participation rate
- **Area** – Rural or Urban

## Project Workflow

### 1. Data Loading
- Imported the dataset using Pandas.
- Examined the dataset structure.

### 2. Data Cleaning
- Checked for missing values.
- Cleaned column names.
- Converted the Date column into datetime format.

### 3. Exploratory Data Analysis (EDA)
- Analyzed unemployment trends over time.
- Studied regional unemployment variations.
- Examined labour participation rates.

### 4. COVID-19 Impact Analysis
- Compared unemployment rates before and during the COVID-19 period.
- Visualized changes using charts.

### 5. Trend Analysis
- Monthly unemployment trend analysis.
- State-wise unemployment comparison.
- Seasonal pattern identification.

### 6. Data Visualization
Created visualizations including:
- Unemployment Rate Over Time
- COVID-19 Impact Analysis
- Top Regions by Unemployment Rate
- Monthly Trend Analysis

---

## Key Findings

- Unemployment rates increased significantly during the COVID-19 pandemic.
- Certain regions consistently recorded higher unemployment rates.
- Monthly fluctuations indicate seasonal employment patterns.
- Labour participation rates were affected during lockdown periods.
- Economic recovery led to gradual reductions in unemployment in several regions.
